# JVM知识

## 一、 字节码、类加载器、虚拟机

### 1.  Java 编程语言跨平台

- Java源代码统一，字节码跨平台不同。
- C++源代码执行跨平台不同。

### 2. 字节码

- 字节码文件  -> 类加载器 -> 创建对象
- 字节码由一些指令组成，主要分成四大类
  - 栈操作指令，局部变量交互指令
  - 程序流程控制指令
  - 对象操作指令，方法调用指令
  - 算术运算以及类型转换指令
- 编译后通过 javap -c xxx 指令可以转化查看相应字节码
- 通过 javap -c -verbose xxx 指令可以查看更多字节码的信息
